# Table

**정적 소량 표** 전용입니다 - React 컴포넌트가 아니라 CSS 클래스로 제공합니다.
정렬·선택·복사·로딩 같은 상호작용이 필요한 데이터 목록은 [DataGrid](/components/datagrid)를 씁니다.
767px 이하에서는 표 대신 카드 리스트로 전환하는 것을 권장합니다.

<div className="demo demo--col">
  <table className="vd-table">
    <thead>
      <tr><th>가맹점</th><th>일자</th><th className="vd-table__num">금액</th></tr>
    </thead>
    <tbody>
      <tr><td>본죽 역삼점</td><td>2026-07-14</td><td className="vd-table__num">9,500원</td></tr>
      <tr><td>맘스터치 선릉점</td><td>2026-07-14</td><td className="vd-table__num">11,200원</td></tr>
      <tr><td>서브웨이 삼성점</td><td>2026-07-13</td><td className="vd-table__num">8,900원</td></tr>
    </tbody>
  </table>
</div>

```html
<table class="vd-table">
  <thead>
    <tr><th>가맹점</th><th>일자</th><th class="vd-table__num">금액</th></tr>
  </thead>
  <tbody>
    <tr><td>본죽 역삼점</td><td>2026-07-14</td><td class="vd-table__num">9,500원</td></tr>
  </tbody>
</table>
```

## 규칙

- 숫자 열에는 `vd-table__num`을 붙입니다 - 우측 정렬 + `tabular-nums`로 자릿수가 맞습니다.
- 헤더는 gray-50 배경 + 13px 텍스트, 행 구분은 1px 보더입니다.
