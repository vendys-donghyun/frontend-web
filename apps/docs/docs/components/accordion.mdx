# Accordion

FAQ나 긴 상세 설명을 접어두는 데 씁니다. 여러 항목을 동시에 열 수 있습니다.

<div className="demo demo--col">
  <Accordion
    defaultOpen={[0]}
    items={[
      { title: '정산은 언제 되나요?', content: '매월 말일 기준으로 익월 10일에 정산됩니다.' },
      { title: '세금계산서는 어떻게 발행하나요?', content: '정산 완료 후 자동으로 발행되며 이메일로 전달됩니다.' },
      { title: '결제 수단을 바꿀 수 있나요?', content: '설정 > 결제 관리에서 언제든 변경할 수 있습니다.' },
    ]}
  />
</div>

```tsx
<Accordion
  defaultOpen={[0]}
  items={[
    { title: '정산은 언제 되나요?', content: '매월 말일 기준으로 익월 10일에 정산됩니다.' },
  ]}
/>
```

## Props

| prop | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `items` | `{ title: string; content: ReactNode }[]` | - | 항목 목록 |
| `defaultOpen` | `number[]` | `[]` | 처음부터 열어둘 항목 인덱스 |
