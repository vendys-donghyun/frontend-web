# Drawer

화면 옆에서 밀려 나오는 측면 패널입니다. 목록을 벗어나지 않고 상세 정보나 필터를
보여줄 때 씁니다. ESC 키·배경 클릭·닫기 버튼으로 닫힙니다.

<div className="demo">
  <DrawerDemo />
</div>

```tsx
const [open, setOpen] = useState(false);

<Drawer
  open={open}
  onClose={() => setOpen(false)}
  title="본죽 역삼점 정산 상세"
  actions={
    <>
      <Button variant="secondary" onClick={() => setOpen(false)}>닫기</Button>
      <Button onClick={confirm}>정산 확정</Button>
    </>
  }
>
  상세 내용
</Drawer>
```

## Props

| prop | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `open` | `boolean` | - | 표시 여부 |
| `onClose` | `() => void` | - | 닫기 요청 (배경 클릭, ESC, 닫기 버튼) |
| `title` | `string` | - | 헤더 제목 |
| `position` | `'left' \| 'right'` | `'right'` | 열리는 방향 |
| `width` | `number` | `400` | 패널 폭(px). 모바일에서는 화면의 85%까지 |
| `actions` | `ReactNode` | - | 하단 고정 버튼 영역 |

## Modal과의 구분

- **Drawer**: 목록 옆에서 상세·필터를 보는 용도 - 컨텍스트를 유지한 채 탐색
- **Modal**: 확정이 필요한 짧은 결정("정산을 확정할까요?") - 모바일에서 바텀시트로 전환
