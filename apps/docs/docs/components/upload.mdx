# FileUpload

파일 선택 버튼과 선택된 파일명 표시입니다. 실제 업로드 처리는 `onChange`에서 합니다.

<div className="demo demo--col" style={{ maxWidth: 420 }}>
  <FileUpload label="사업자등록증" help="PDF 또는 이미지, 10MB 이하" accept=".pdf,image/*" />
  <FileUpload label="정산 자료" buttonText="파일 추가" multiple />
</div>

```tsx
<FileUpload
  label="사업자등록증"
  help="PDF 또는 이미지, 10MB 이하"
  accept=".pdf,image/*"
  onChange={(e) => upload(e.target.files)}
/>
```

## Props

| prop | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `label` / `help` / `error` | `string` | - | 필드 공통 문법 |
| `buttonText` | `string` | `'파일 선택'` | 버튼 문구 |

그 외 `input[type=file]` 기본 속성(`accept`, `multiple` 등)을 그대로 받습니다.
