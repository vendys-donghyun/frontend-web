# Input · Select

라벨은 입력창 위에 둡니다. 플레이스홀더로 라벨을 대체하지 않습니다.

## Input

<div className="demo demo--col" style={{ maxWidth: 360 }}>
  <Input label="이메일" placeholder="name@vendys.co.kr" help="회사 이메일을 입력해 주세요" />
  <Input label="사업자등록번호" defaultValue="123-45-678" error="사업자등록번호는 10자리 숫자입니다. 다시 확인해 주세요." />
  <Input label="회사명" defaultValue="현대벤디스" readOnly />
</div>

```tsx
<Input label="이메일" placeholder="name@vendys.co.kr" help="회사 이메일을 입력해 주세요" />
<Input label="사업자등록번호" error="사업자등록번호는 10자리 숫자입니다. 다시 확인해 주세요." />
```

## Select

입력창과 같은 프레임을 씁니다. 모바일에서는 네이티브 드롭다운이 뜹니다.

<div className="demo demo--col" style={{ maxWidth: 360 }}>
  <Select label="정산 주기" defaultValue="month">
    <option value="week">매주</option>
    <option value="month">매월</option>
  </Select>
</div>

## Props (공통)

| prop | 타입 | 설명 |
|---|---|---|
| `label` | `string` | 입력창 위 13px 라벨 |
| `help` | `string` | 아래 12px 도움말 |
| `error` | `string` | 오류 문구. 설정 시 오류 상태로 표시, `aria-invalid` 적용 |

그 외 `input` / `select` 기본 속성을 모두 받습니다.

## 오류 문구 규칙

- 원인과 해결을 구체적으로 씁니다: "사업자등록번호는 10자리 숫자입니다" (O), "입력값이 올바르지 않습니다" (X)
