# Radio

여러 선택지 중 하나만 고를 수 있을 때 씁니다. 같은 그룹은 `name`으로 묶습니다.

<div className="demo demo--col" style={{ gap: 4 }}>
  <Radio name="cycle-demo" defaultChecked>매월 정산</Radio>
  <Radio name="cycle-demo">매주 정산</Radio>
  <Radio name="cycle-demo" disabled>일 정산 (준비 중)</Radio>
</div>

```tsx
<Radio name="cycle" value="month" checked={cycle === 'month'} onChange={onChange}>매월 정산</Radio>
<Radio name="cycle" value="week" checked={cycle === 'week'} onChange={onChange}>매주 정산</Radio>
```

## Props

`input` 기본 속성(`name`, `value`, `checked`, `onChange`, `disabled` 등)을 그대로 받고
`children`이 라벨입니다. `ref` 전달을 지원합니다.

## Guidelines

- 선택지가 5개를 넘으면 Radio 나열 대신 [Select](./select.mdx)를 검토합니다.
