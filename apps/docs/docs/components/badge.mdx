# Badge

상태·분류 표시용입니다. 행동(버튼) 용도로 쓰지 않습니다.
연한 배경 + 진한 글자(weak 톤)가 기본형입니다.

<div className="demo">
  <Badge tone="success">정산 완료</Badge>
  <Badge>대기</Badge>
  <Badge tone="warning">확인 필요</Badge>
  <Badge tone="error">실패</Badge>
</div>

```tsx
<Badge tone="success">정산 완료</Badge>
<Badge>대기</Badge>
<Badge tone="warning">확인 필요</Badge>
<Badge tone="error">실패</Badge>
```

## Props

| prop | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `tone` | `'success' \| 'neutral' \| 'warning' \| 'error'` | `'neutral'` | 성공(초록) / 중립(회색) / 주의(주황) / 실패(빨강) |

성공 톤은 별도 색이 아니라 primary(#43A047)를 재사용합니다.
