# Tabs

활성 탭은 primary 굵은 글자 + 2px 언더라인으로 표시합니다.
모바일에서는 가로 스크롤됩니다.

<div className="demo demo--col">
  <TabsDemo />
</div>

```tsx
const [tab, setTab] = useState('all');

<Tabs
  items={[
    { value: 'all', label: '전체' },
    { value: 'wait', label: '정산 대기' },
    { value: 'done', label: '정산 완료' },
  ]}
  value={tab}
  onChange={setTab}
/>
```

## Props

| prop | 타입 | 설명 |
|---|---|---|
| `items` | `{ value: string; label: string }[]` | 탭 목록 |
| `value` | `string` | 활성 탭 value (제어 컴포넌트) |
| `onChange` | `(value: string) => void` | 탭 클릭 시 호출 |

`role="tablist"` / `role="tab"` / `aria-selected`가 자동 적용됩니다.
