import { Tooltip, Button, Icon } from '@vendys/ui';

# Tooltip

hover/focus 시 한 줄 보조 설명을 보여줍니다.
터치 환경에서는 노출이 제한적이니 **필수 정보는 툴팁에 넣지 않습니다.**

<div className="demo" style={{ paddingTop: 40 }}>
  <Tooltip label="정산 내역을 내려받습니다">
    <Button variant="secondary">다운로드</Button>
  </Tooltip>
  <Tooltip label="영업일 기준 3일" position="bottom">
    <span style={{ display: 'inline-flex', color: 'var(--vd-text-sub)' }}><Icon name="info" size={16} /></span>
  </Tooltip>
</div>

```tsx
<Tooltip label="정산 내역을 내려받습니다">
  <Button variant="secondary">다운로드</Button>
</Tooltip>
```

## Props

| prop | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `label` | `string` | - | 말풍선 문구 (한 줄) |
| `position` | `'top' \| 'bottom' \| 'left' \| 'right'` | `'top'` | 표시 위치 |
