# Select

입력창과 같은 프레임의 커스텀 셀렉트입니다. 단일 선택이 기본이고,
`multiple`(다중 선택)과 `searchable`(검색) 옵션을 조합할 수 있습니다.

## Single Select

<div className="demo demo--col" style={{ maxWidth: 360 }}>
  <Select
    label="정산 주기"
    options={[
      { value: 'week', label: '매주' },
      { value: 'month', label: '매월' },
      { value: 'quarter', label: '분기', disabled: true },
    ]}
  />
</div>

```tsx
<Select
  label="정산 주기"
  value={cycle}
  onChange={setCycle}
  options={[
    { value: 'week', label: '매주' },
    { value: 'month', label: '매월' },
  ]}
/>
```

## Multiple

체크박스로 여러 개를 고릅니다. 선택해도 패널이 닫히지 않고,
선택된 라벨이 트리거에 이어서 표시됩니다.

<div className="demo demo--col" style={{ maxWidth: 360 }}>
  <Select
    label="정산 대상 지점"
    multiple
    placeholder="지점 선택"
    options={[
      { value: 'yeoksam', label: '본죽 역삼점' },
      { value: 'seolleung', label: '맘스터치 선릉점' },
      { value: 'samsung', label: '서브웨이 삼성점' },
      { value: 'daechi', label: '한솥도시락 대치점' },
    ]}
  />
</div>

```tsx
<Select multiple value={stores} onChange={setStores} options={storeOptions} />
```

## Searchable

옵션이 많을 때 패널 상단에서 검색으로 거릅니다. multiple과 함께 쓸 수 있습니다.

<div className="demo demo--col" style={{ maxWidth: 360 }}>
  <Select
    label="가맹점"
    searchable
    placeholder="가맹점 선택"
    options={[
      { value: '1', label: '본죽 역삼점' },
      { value: '2', label: '맘스터치 선릉점' },
      { value: '3', label: '서브웨이 삼성점' },
      { value: '4', label: '한솥도시락 대치점' },
      { value: '5', label: '김밥천국 역삼2호점' },
      { value: '6', label: '샐러디 테헤란점' },
    ]}
  />
</div>

## Props

| prop | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `options` | `{ value, label, disabled? }[]` | - | 옵션 목록 |
| `multiple` | `boolean` | `false` | 체크박스 다중 선택. `value`/`onChange`가 `string[]`이 된다 |
| `searchable` | `boolean` | `false` | 패널 상단 검색 입력 |
| `value` / `onChange` | `string` 또는 `string[]` | - | 미지정 시 내부 상태로 동작 (비제어) |
| `placeholder` | `string` | `'선택'` | 미선택 시 표시 문구 |
| `label` / `help` / `error` | `string` | - | 필드 공통 문법 |

## Guidelines

- 선택지가 5개 이하면 [Radio](./radio.mdx)(단일)나 [Checkbox](./checkbox.mdx)(다중) 나열을 먼저 검토합니다.
- 행동을 실행하는 용도(더 보기 메뉴)는 [Menu](./menu.mdx)를 씁니다 - Select는 값을 고르는 폼 컨트롤입니다.
