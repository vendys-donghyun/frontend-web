# Toast

작업 결과를 짧게 알립니다. 문구는 과거형 한 문장입니다: "저장되었습니다".
하단 중앙에 띄우고 3초 뒤 제거합니다.

<div className="demo">
  <Toast>저장되었습니다</Toast>
  <Toast tone="error">저장하지 못했습니다</Toast>
  <Toast tone="warning">3건은 확인이 필요합니다</Toast>
  <Toast tone="info">새 정산 내역이 있습니다</Toast>
  <ToastDemo />
</div>

```tsx
// 하단 중앙 고정이 필요하면 viewport 컨테이너 안에 렌더링
<div className="vd-toast-viewport">
  <Toast>저장되었습니다</Toast>
</div>
```

## Props

| prop | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `tone` | `'success' \| 'error' \| 'warning' \| 'info'` | `'success'` | 상태 점 색: 초록 / 빨강 / 주황 / 파랑 |

`role="status"`가 적용되어 스크린리더에 자동으로 읽힙니다.

## 규칙

- 확인 버튼이 필요한 내용(되돌릴 수 없는 실패 등)은 토스트가 아니라 모달을 씁니다.
