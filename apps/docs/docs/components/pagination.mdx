# Pagination

목록·표 하단의 페이지 이동입니다. 7페이지를 넘으면 현재 페이지 주변만 보여주고
나머지는 줄임표로 접습니다.

<div className="demo">
  <PaginationDemo />
</div>

```tsx
const [page, setPage] = useState(1);

<Pagination page={page} totalPages={12} onChange={setPage} />
```

## Props

| prop | 타입 | 설명 |
|---|---|---|
| `page` | `number` | 현재 페이지 (1부터) |
| `totalPages` | `number` | 전체 페이지 수 |
| `onChange` | `(page: number) => void` | 페이지 클릭 시 호출 |

현재 페이지는 `aria-current="page"`로 표시됩니다.
