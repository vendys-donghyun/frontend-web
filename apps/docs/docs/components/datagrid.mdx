# DataGrid

조회형 데이터 그리드입니다. vone-web의 ag-Grid(ClientSideTable) 사용 패턴 중
가장 빈도가 높은 것들 - 셀 포맷(빈값 '-', 숫자 콤마, 날짜), 상태 배지, 정렬,
행 선택, 로딩/빈 상태 - 을 디자인 토큰 기반으로 다시 구현했습니다.

<div className="demo demo--col">
  <DataGridDemo />
</div>

```tsx
const columns: DataGridColumn<SettlementRow>[] = [
  { field: 'store', headerName: '가맹점' },
  {
    field: 'status', headerName: '상태', width: 110, align: 'center',
    render: (value) => <Badge tone={STATUS_TONE[value]}>{value}</Badge>,
  },
  { field: 'settledAt', headerName: '정산일', type: 'date', width: 120 },
  { field: 'amount', headerName: '금액(원)', type: 'number', width: 130 },
];

<DataGrid
  columns={columns}
  rows={rows}
  rowKey={(row) => row.id}
  selectable="multi"
  onSelectionChange={(keys, rows) => setSelected(keys)}
  loading={isFetching}
/>
```

## 정렬 · 필터 · 컬럼 순서 변경

기본 데모에서 바로 써볼 수 있습니다:

- **정렬**: 헤더 클릭 - 오름 → 내림 → 해제 순환
- **필터**: 헤더의 깔때기 아이콘 - 값 목록에서 체크로 거릅니다 (검색·전체 선택·초기화 포함, ag-Grid의 셋 필터 방식). 필터가 걸린 컬럼은 아이콘이 초록색이 됩니다
- **컬럼 순서 변경**: 헤더를 드래그해서 원하는 위치에 놓습니다 (고정 컬럼은 같은 고정 그룹 안에서만)

## 셀 범위 선택 · 복사

셀을 누른 채 드래그하면 범위가 선택됩니다. 선택 후:

- **Ctrl/Cmd + C** - 선택 영역을 탭 구분 텍스트로 복사 (엑셀에 그대로 붙습니다)
- **우클릭** - 셀 값 복사 / 선택 영역 복사 / 행 복사 메뉴
- **ESC** - 선택 해제

## 컬럼 고정 · 셀 복사

컬럼이 많아 가로 스크롤이 생기는 화면에서는 `pinned`로 좌/우 컬럼을 고정합니다
(식별 컬럼은 왼쪽, 액션 컬럼은 오른쪽 - vone-web 관례와 동일). 고정 컬럼은 `width` 지정이 필수이고,
선택 체크박스 열은 자동으로 맨 왼쪽에 고정됩니다.
셀을 **우클릭하면 복사 메뉴**(셀 값 / 행 전체, 탭 구분)가 열립니다.

```tsx
{ field: 'store', headerName: '가맹점', width: 150, pinned: 'left' },
{ field: 'actions', headerName: '액션', width: 90, pinned: 'right', sortable: false, render: ... },

<DataGrid columns={columns} rows={rows} rowKey={(r) => r.id} minWidth={1240} />
```

붙여넣기(셀 편집·엑셀 대량 입력)는 지원하지 않습니다 - 편집 영역이라 ag-Grid를 유지합니다.

## 컬럼 정의 (DataGridColumn)

| 속성 | 타입 | 설명 |
|---|---|---|
| `field` | `string` | 행 객체의 키 |
| `headerName` | `string` | 헤더 문구 |
| `width` | `number` | px 고정폭. 없으면 남은 폭 균등 분배 |
| `type` | `'text' \| 'number' \| 'date' \| 'datetime'` | number는 콤마+우측정렬, date는 YYYY-MM-DD. 빈값은 '-' |
| `align` | `'left' \| 'center' \| 'right'` | 기본 left (number는 right) |
| `sortable` | `boolean` | 기본 true. 헤더 클릭 → 오름/내림/해제 순환 |
| `filterable` | `boolean` | 기본 true. 값 목록 필터 |
| `pinned` | `'left' \| 'right'` | 컬럼 고정. `width` 지정 필수, 가로 스크롤과 함께 사용 |
| `render` | `(value, row) => ReactNode` | 커스텀 셀 (배지, 링크, 버튼) |
| `valueGetter` | `(row) => unknown` | 표시·정렬 값 계산 |

## Props

| prop | 타입 | 설명 |
|---|---|---|
| `columns` / `rows` / `rowKey` | - | 필수 |
| `selectable` | `'single' \| 'multi'` | 체크박스 열 추가. multi는 헤더 전체 선택 |
| `selectedKeys` / `onSelectionChange` | - | 선택 제어 (미지정 시 내부 상태) |
| `onRowClick` | `(row) => void` | 행 클릭 |
| `onSortChange` | `(sort) => void` | 정렬 변경 알림 (서버 정렬 연동용) |
| `loading` | `boolean` | 스피너 오버레이 |
| `emptyMessage` | `string` | 기본 "조회된 데이터가 없습니다" |
| `density` | `'standard' \| 'compact'` | 행 48 / 36 |
| `height` | `number \| string` | 지정 시 헤더 고정 세로 스크롤 |
| `minWidth` | `number` | 표 최소 폭. 컨테이너보다 크면 가로 스크롤 |
| `enableCopy` | `boolean` | 기본 true. 셀 우클릭 복사 메뉴 |
| `columnReorder` | `boolean` | 기본 true. 헤더 드래그로 컬럼 순서 변경 |
| `cellSelection` | `boolean` | 기본 true. 셀 드래그 범위 선택 + Ctrl/Cmd+C |

## 범위

이 컴포넌트는 **조회형 클라이언트 그리드**(vone-web 사용량의 대다수)를 대상으로 합니다.
인라인 셀 편집, 서버사이드 로우 모델, 엑셀 붙여넣기(대량 입력)가
필요한 화면은 ag-Grid를 유지하는 하이브리드 전략을 권장합니다.
