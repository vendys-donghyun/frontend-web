# Figma Kit

모든 컴포넌트를 한 보드에 모았습니다. **전체 Figma로 복사**를 누른 뒤
Figma에서 붙여넣으면(Cmd+V) 편집 가능한 도형·텍스트 레이어로 들어갑니다.

- 개별 요소만 필요하면 각 컴포넌트 문서의 데모에 마우스를 올리면 나오는 **Figma로 복사** 버튼을 쓰세요.
- 텍스트가 올바른 서체로 보이려면 Figma를 쓰는 PC에 **Pretendard**가 설치돼 있어야 합니다.
- 붙여넣은 결과는 시안용 사본입니다 - 배리언트가 잡힌 정식 Figma 컴포넌트가 아닙니다.
