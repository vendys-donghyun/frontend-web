# Spacing · Shape

간격, 모서리, 그림자, 컨트롤 높이의 기준입니다.

## Spacing (4px Grid)

모든 간격은 4px의 배수입니다. 아래 값 외의 간격(예: 10px, 18px)은 쓰지 않습니다.
막대 길이가 실제 간격 크기입니다.

<div className="demo demo--col">
  <SpacingScale />
</div>

```css
padding: var(--vd-space-5);      /* 20px - 데스크톱 카드 패딩 */
gap: var(--vd-space-2);          /* 8px */
```

## Radius

기본은 8px이고, 요소가 클수록 모서리도 비례해서 커집니다.

<div className="demo">
  <RadiusScale />
</div>

## Shadow

그림자는 3단계뿐입니다. 붙어 있는 요소(카드 기본형)는 그림자 없이 보더만 씁니다.

<div className="demo">
  <ShadowScale />
</div>

## Control Height

버튼·인풋 높이는 3단입니다. 기본은 md(40px)이고, 모바일 주요 동작은 lg(48px)를 씁니다.

<div className="demo">
  <ControlScale />
</div>

- 모바일 터치 대상은 최소 44px을 지킵니다. lg(48px)는 자동으로 충족하고,
  작은 요소는 여백을 포함해 44px을 확보합니다.

## Breakpoint

기준점은 하나입니다. 767px 이하 Mobile, 768px 이상 Desktop.

```css
@media (max-width: 767px) { /* Mobile */ }
```
