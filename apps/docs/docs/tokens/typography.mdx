# 타이포그래피

폰트는 **Pretendard**를 사용하고, 미설치 환경을 위해 시스템 폰트 폴백을 포함합니다.

```css
font-family: var(--vd-font);
/* Pretendard, -apple-system, BlinkMacSystemFont, system-ui,
   Roboto, 'Helvetica Neue', 'Segoe UI', 'Apple SD Gothic Neo',
   'Noto Sans KR', 'Malgun Gothic', sans-serif */
```

## 스케일

Desktop / Mobile 두 스케일만 정의합니다 (태블릿 없음).

| 역할 | Desktop | Mobile | 굵기 |
|---|---|---|---|
| Display | 32 / 42 | 26 / 36 | 700 |
| Heading 1 | 24 / 34 | 22 / 30 | 700 |
| Heading 2 | 20 / 28 | 18 / 26 | 600 |
| Heading 3 | 17 / 25 | 16 / 24 | 600 |
| Body | 15 / 23 | 15 / 23 | 400 |
| Body small | 13.5 / 20 | 13.5 / 20 | 400 |
| Caption | 12.5 / 18 | 12 / 17 | 400 |

```ts
import { typography } from '@vendys/tokens';

typography.desktop.h1; // { size: 24, lineHeight: 34, weight: 700 }
```

## 규칙

- 숫자를 세로로 비교하는 곳(표의 금액 열 등)은 `font-variant-numeric: tabular-nums`를 적용합니다.
- 본문 텍스트 색은 `--vd-text`(gray-900), 보조 텍스트는 `--vd-text-sub`(gray-600)입니다.
