# 타이포그래피

폰트는 **Pretendard**를 사용합니다. 자간은 -0.2px이고, 미설치 환경을 위해
시스템 폰트 폴백을 포함합니다.

```css
font-family: var(--vd-font);
/* 'Pretendard Variable', 'Pretendard', -apple-system, BlinkMacSystemFont,
   'Apple SD Gothic Neo', 'Malgun Gothic', sans-serif */
```

## 스케일

Desktop / Mobile 두 스케일을 정의합니다.
아래 견본은 실제 토큰 값으로 Desktop / Mobile 크기를 각각 렌더링한 것입니다.

<div className="demo demo--col">
  <TypeScale />
</div>

```ts
import { typography } from '@vendys/tokens';

typography.scale.title; // { desktop: 24, mobile: 22, weight: 700 }
```

## 규칙

- 역할은 위 7개가 전부입니다. 중간 크기(15px, 19px 등)를 임의로 만들지 않습니다.
- 본문 텍스트 색은 `--vd-text`(#212121), 보조 텍스트는 `--vd-text-sub`(#616161)입니다.
- 숫자를 세로로 비교하는 곳(표의 금액 열 등)은 `font-variant-numeric: tabular-nums`를 적용합니다.
