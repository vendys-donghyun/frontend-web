# 색상

Primary는 `#43A047` 하나입니다. Material Green 램프에서 파생한 hover/active와
Material Gray 램프를 함께 사용합니다.

## Primary

<div className="demo">
  <Swatch color="#43a047" name="primary" hex="#43A047" />
  <Swatch color="#388e3c" name="primary-hover" hex="#388E3C" />
  <Swatch color="#2e7d32" name="primary-active" hex="#2E7D32" />
  <Swatch color="#e8f5e9" name="primary-tint" hex="#E8F5E9" />
  <Swatch color="#c8e6c9" name="primary-tint-hover" hex="#C8E6C9" />
</div>

```css
color: var(--vd-primary);            /* #43A047 */
background: var(--vd-primary-tint);  /* #E8F5E9 - weak 버튼, 배지 배경 */
```

성공(success) 상태는 별도 색을 만들지 않고 primary를 재사용합니다.
