## Guidelines

**Do**

- 초록(#43A047)은 주요 버튼 · 활성 상태 · 포커스에만
- 비동기 동작 버튼에는 loading 상태를 넣고 버튼 폭 유지
- 간격은 4px 그리드, 모서리 기본 8px
- 오류 문구는 원인과 해결책을 구체적으로

**Don't**

- 새로운 초록 값 생성 금지 (과거 #46B476, #0BB656 등)
- 순수 검정(#000) 본문, 임의 회색 금지
- 한 화면에 Primary 버튼 2개 이상 금지
- 장식용 그림자 · 그라디언트, UI 이모지 금지
