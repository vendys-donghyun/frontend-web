# AI 프롬프트

이 디자인 시스템 전체를 **AI가 읽을 수 있는 한 개의 문서**(`DESIGN.md`)로 제공합니다.
아래 전체 복사 버튼으로 복사해서 프롬프트에 붙여넣으면, AI가 토큰과 규칙대로 UI를 만듭니다.

## 사용법

- **채팅형 AI** (Claude, ChatGPT 등): 아래 문서를 붙여넣고 "이 디자인 기준으로 ○○ 화면을 만들어줘"
- **레포 안의 AI 도구** (Claude Code 등): 복사할 필요 없이 `packages/tokens/DESIGN.md`를 컨텍스트로 지정
- 문서 원본은 `@vendys/tokens` 패키지 안에 있고, 이 페이지는 그 원본을 그대로 보여줍니다 - 사본이 아니라서 어긋나지 않습니다
