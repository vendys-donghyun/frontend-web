# AI 프롬프트

<div className="ai-prompt-layout">
<div className="ai-prompt-layout__docs">

이 디자인 시스템 전체를 **AI가 읽을 수 있는 한 개의 문서**(`DESIGN.md`)로 제공합니다.
오른쪽 문서를 복사해서 프롬프트에 붙여넣으면, AI가 토큰과 규칙대로 UI를 만듭니다.

### 사용법

- **채팅형 AI** (Claude, ChatGPT 등): 오른쪽 문서를 복사해 붙여넣고 요청을 이어서 씁니다
- **레포 안의 AI 도구** (Claude Code 등): 복사할 필요 없이 `packages/tokens/DESIGN.md`를 컨텍스트로 지정합니다

### 예시 프롬프트

```text
(DESIGN.md 붙여넣기)

이 디자인 기준으로 가맹점 정산 내역 화면을 만들어줘.
상단에 기간 필터, 아래에 정산 목록 표.
```

### 문서에 담긴 것

- **토큰 요약** - 색상, 타이포그래피, 간격, radius, 그림자, 모션
- **규칙** - 색 역할 제한, 임의 값 금지, 과거 초록 값 사용 금지
- **컴포넌트 20종 계약** - 크기·색·상태(hover/loading/disabled)까지
- **Do / Don't, 반응형, 상태, 보이스 & 톤**
